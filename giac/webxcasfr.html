Documentation, exemples de commandes :
  • Algèbre : factor(x^4-1); cfactor(x^2+1); normal((x+1)^4); simplify(sin(3x)/sin(x)); gcd(x^4-1,x^3-1)
  • Résoudre une ou des équations : solve(x^2-3*x+2=0); csolve(x^2=2*i); solve([x+y=1,x-y=3],[x,y])
  • Analyse : f(x):=sin(x^2):; f(sqrt(pi)); f'(2); f'(y)
    int(1/(x^4-1)); int(1/(x^4+1)^4,x,0,+infinity); limit(sin(x)/x,x=0); series(sin(x),x=0,5);
  • Algèbre linéaire : A:=[[1,2],[3,4]]; inv(A); det(A-x*idn(A)); A[0,0]; rref(A); eigenvalues(A); eigenvectors(A);
  • Graphes (pour les graphes 3d, vous pouvez changer de point de vue en utilisant les touches de déplacement, taper q pour quitter le mode interactif) : plot([sin(x),x-x^3/3!],x=-3..3,color=[red,blue])
    plotfunc(x^2-y^2,[x=-2..2,y=-2..2]); plane(z=0,color=cyan+filled);

Webxcas est une interface de demo du système de calcul formel Giac/Xcas compilé en javascript par emscripten. Il n'a pas besoin de serveur, il s'exécute localement (avec le moteur javascript de votre navigateur qui doit être récent, par exemple Firefox 19 ou Chrome) à partir du code du CAS (10M) qui est téléchargé une fois. Le prix à payer pour cette simplicité est la vitesse, le code est au moins 2 fois plus lent, parfois plus de 100 fois plus lent (par exemple pour le calcul de bases de Groebner comme cyclic7). La plupart du temps, cela n'est pas vraiment genant (ce qui justifierait une interface plus conviviale...), mais si vous devez exécuter un gros calcul, il vaut mieux installer Xcas!
